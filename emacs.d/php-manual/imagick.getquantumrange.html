Imagick::getQuantumRange

(PECL imagick 2.0.0)

Imagick::getQuantumRange — Returns the Imagick quantum range

Description

array Imagick::getQuantumRange ( void )

Returns the Imagick quantum range as a string.

Return Values

Returns the Imagick quantum range as a string.

Errors/Exceptions

Throws ImagickException on error.
