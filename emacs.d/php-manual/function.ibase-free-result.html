ibase_free_result

(PHP 4, PHP 5)

ibase_free_result — Free a result set

Description

bool ibase_free_result ( resource $result_identifier )

Frees a result set.

Parameters

result_identifier

A result set created by ibase_query() or ibase_execute().

Return Values

Returns TRUE on success or FALSE on failure.
