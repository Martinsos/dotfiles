(PECL trader >= 0.2.0)

trader_linearreg_intercept — Linear Regression Intercept

Description

array trader_linearreg_intercept ( array $real [, integer $timePeriod ] )

Parameters

real

Array of real values.

timePeriod

Number of period. Valid range from 2 to 100000.

Return Values

Returns an array with calculated data or false on failure.
