Xcom::getLastResponseInfo

(PECL xcommerce >= 1.0.0)

Xcom::getLastResponseInfo — Gets the last response HTTP information

Description

public array Xcom::getLastResponseInfo ( void )

Get the last response information such as HTTP response code, latencies, and redirect URL's.

Parameters

This function has no parameters.

Return Values

Returns an array containing HTTP response information.
