PDF_end_layer

(PECL pdflib >= 2.0.0)

PDF_end_layer — Deactivate all active layers

Description

bool PDF_end_layer ( resource $pdfdoc )

Deactivates all active layers. Returns TRUE on success or FALSE on failure.

This function requires PDF 1.5.
